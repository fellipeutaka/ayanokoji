Drizzle is a simple, fast, type-safe and lightweight ORM for TypeScript.
It supports MySQL, SQLite, and PostgreSQL.
Also, it supports all platform-specific, tcp, http, websocket based drivers,
and every major serverful and serverless runtime.

[Documentation](https://orm.drizzle.team/docs/overview)

## Usage

```package-install
npx ayanokoji drizzle init
```
